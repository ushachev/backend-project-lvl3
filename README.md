# Web page downloader
Downloads a web page for local viewing.

## Setup

```sh
$ make install
```

## Application launch

```sh
$ make publish
$ sudo npm link
```
this will create a symlink to the package after that it will become possible to launch application by typing

```sh
$ page-loader
```
